# PII Scanner - PC 개인정보 탐지 솔루션

## 프로젝트 개요
Windows PC에서 파일 내 개인정보(PII)를 탐지하여 관리자에게 보고하는 상용 솔루션.
서버-클라이언트 아키텍처로 수만 대 규모의 PC를 관리한다.
구독형 라이선스 모델로 기업에 판매한다.

## 배경 (왜 만드는가)
개발자는 약 50,000대 PC에서 기존 개인정보 탐지 솔루션을 15년간 운영했다.
기존 솔루션의 핵심 문제점:
- 높은 오탐/과탐률: 정규식만으로는 한국어 PII를 정확히 구분할 수 없음
- 사용자 업무 방해: 풀스캔 시 CPU 급증, 반복 팝업에 대한 피로도
- 비정형 PII 미탐지: 이름, 주소, 이름+주소 조합 등을 찾지 못함
- 실시간 탐지 불가: 파일 생성/변경 시점에 즉시 감지하지 못함
- DRM 파일 탐지 불가
- Windows만 지원, 증분 스캔 미지원, 관리 인사이트 부족

## 핵심 목표
1. 오탐 및 과탐률을 획기적으로 낮출 것
2. 사용자 업무를 방해하지 않을 것
3. 실시간 파일 감지를 지원할 것
4. 비정형 PII(이름, 주소 등)도 탐지할 것
5. GPU 없이 CPU만으로 동작할 것 (기업 PC 환경)
6. 상용 제품 수준의 보안과 라이선스 체계를 갖출 것

## 기술 방향 (참고용 — 더 나은 대안이 있으면 제안해도 됨)
- 클라이언트: Rust 선호 (성능, 메모리 안전성, 단일 바이너리)
- NER: KoELECTRA 계열 검토 중 (ONNX 양자화로 CPU 추론)
- 탐지 전략: 정규식+체크섬 1차, NER은 의심 텍스트에만 적용하는 하이브리드 방식
- 서버: 적합한 기술 스택
